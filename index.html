1. Abstract

Rida Network is a decentralized-logic utility platform designed to lower the barrier to entry for digital asset participation. Utilizing Proof-of-Presence (PoP), Rida allows users to generate value without hardware costs.

2. Proof-of-Presence (PoP)

Unlike energy-heavy mining, PoP rewards active daily synchronization. Users initialize their node every 24 hours to secure 0.2 RIDA yield.

3. Security & Verification

To prevent bot inflation, only Verified Nodes can execute P2P transfers. Verification is granted manually by administrators based on node consistency.

4. P2P Economy

  • Min Transfer: 10 RIDA
  • Fee Model: 0% Utility phase
  • Hold Rule: Balance < 5 RIDA triggers auto-unverify
